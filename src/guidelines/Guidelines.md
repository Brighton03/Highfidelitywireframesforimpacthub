# Design System Guidelines

## Brand Identity
* **App Name:** ImpactHub
* **Logo:** Text-based "ImpactHub". "Impact" in Dark Grey (#2C3E50), "Hub" in Sage Green (#779F8D).
* **Primary Brand Color:** #779F8D (Sage Green) - Used for primary buttons, active states, progress bars, and success indicators.
* **Secondary Brand Color:** #2C3E50 (Dark Grey/Blue) - Used for headings, navigation bars, sidebars, and primary text.
* **Background Color:** #F5F7FA (Light Grey/White) - Used for app backgrounds to ensure readability.
* **Surface Color:** #FFFFFF (White) - Used for cards, modals, and input fields.
* **Error Color:** #E57373 (Soft Red) - Used for error states, destructive actions, and alerts.

## Typography
* **Font Family:** Inter or Roboto (Modern Sans-Serif).
* **Headings:** Bold, #2C3E50.
* **Body Text:** Regular, #2C3E50, 14px or 16px. High legibility.

## Layout Principles
* **Desktop Web:** 1440px width.
    * **Volunteers:** Top horizontal navigation bar.
    * **Coordinators/Admins:** Left vertical sidebar navigation for complex management tools.
* **Mobile App:** 375px width (iPhone 14/15). Bottom tab bar navigation.
* **Spacing:** Use an 8px grid system. 16px padding for mobile containers, 24px-32px for desktop.
* **Corner Radius:** 8px for cards, buttons, and modals.

## Component States
* **Buttons:**
    * *Primary:* Solid #779F8D fill, White text.
    * *Secondary:* Transparent background, #2C3E50 border.
    * *Disabled:* Light Grey background (#CFD8DC), text opacity 50%.
* **Status Badges:**
    * *Success/Confirmed:* Green pill.
    * *Waitlist/Pending:* Orange pill.
    * *Error/Flagged:* Red pill.
